新手指引插件的使用格式为：
$.guideInit(0.2) 初始化，会创建一个蒙版，0.2为透明度参数，可有可无
$.showGuide([
			{
				'el':'#t1',
				'title':"新手指引",
				'content':"新手指引内容1"
			},
			{
				'el':'#t2',
				'title':"新手指引",
				'content':"新手指引内容2"
			}]);
备注：el即为dom元素，类似于jquery的选择器,即放入需要创立新手指引高亮显示的区域
