#GTPS
###简介
GTPS全称Gimmay Transaction Processing Systems，是一个用于处理和管理吉美集团各部门工作中所需要处理的业务流程的一套业务系统。

###开发/运行环境软件需求
- Mysql 5.7.*
- PHP ~7.0.18
- Apache ~2.4.25
- nodejs >=8.0.0
- git ^2.0.0
- Redis ^3.0.0
